# Instructional Integrity Studio UI

## What this is

This is a Next.js App Router UI scaffold built around the ideal product journey for the instructional integrity system:

1. Understand the framework
2. Choose a primary path
3. Complete one focused task
4. Review evidence
5. Move into production packages

## What is functional right now

- Complete homepage UI
- Stable information architecture
- Package-to-screen mapping
- Evaluator interface mock with visible evidence states
- Tailwind-based styling
- TypeScript project structure

## What is not wired yet

- Real evaluator execution
- File upload pipeline
- Persistent state
- Authentication
- API integrations
- Training workflow backend

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
